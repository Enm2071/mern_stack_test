## Installation and running the app

Make sure you have Node.js installed on your computer.

1. Ensure you have [Node.js](https://nodejs.org/) version 14.20
2. Run the following command to install the dependencies:

   ```shell
   npm install
   
3. Start the Development Server: After the dependencies are installed, start the development server by running the following command:

   ```shell
   npm start
   ```

Scripts
-------

In the project, you can run the following scripts:

- `npm run build`: Compile the application in production mode.
- `npm start`: Start the development server and open the application in the browser.
- `npm run format`: Format the code using Prettier.
- `npm run lint`: Run the linter (ESLint) to check the code.
- `npm test`: Run the test cases (Jest).
- `npm run test:watch`: Run the test cases in watch mode.
- `npm run test:update`: Update the test snapshots (Jest).

Make sure to execute the appropriate commands based on your needs and development environment.

## How to use the app

1. Open the app in the browser.
2. Click on Sign-up button to register a new user.
3. Enter the details and click on Sign-up button.
4. Click on Login button to login with the registered user.
5. Enter the details and click on Login button.
6. Go to Store page to view the list of products.
7. Click on Add to Cart button to add the product to cart.
8. Click on Cart button to view the list of products added to cart.
9. Click on Checkout button to place the order.
10. Click on Orders button to view the list of orders placed.
11. Click on the order to view the order details.
12. On the order details page, click on Cancel button to cancel the order.
13. On the order details page, click on Pay button to pay for the order.
10. Click on Logout button to logout from the app.
